Edit the navigation and hero section:

  1. Press Cmd + C to activate React Grab
  2. Select the navbar and change colors, padding, or spacing
  3. Select the hero section and edit the gradient, typography
  4. Press T to track elements before editing
  5. Press C to copy all visual changes
  6. Give the output to your AI to apply the changes!
