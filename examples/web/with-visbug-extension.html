Instructions

  1. Click VisBug extension icon in browser toolbar to activate VisBug
  2. Click on any element to select it
  3. Use VisBug tools (paint bucket for colors, 'A' for text, etc.)
  4. Press Cmd + C to activate React Grab
  5. Press T to track the element
  6. Press C to copy visual changes
  7. Paste to your AI agent
