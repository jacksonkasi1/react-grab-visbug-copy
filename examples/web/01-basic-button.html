Example 1: Basic Button

Edit this button with VisBug and copy the changes!

How to use:

  1. Click VisBug icon in browser toolbar to activate VisBug
  2. Press Cmd + C to activate React Grab
  3. Click on the button to select it
  4. Use VisBug tools to change color, font-size, etc.
  5. Press T to track changes
  6. Press C to copy changes
